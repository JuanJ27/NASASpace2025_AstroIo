## Core Files

### server.js (7.5 KB)
**Backend server implementation**
- Express web server
- Socket.IO real-time communication
- Game world management (2000x2000 pixels)
- 200 orbs with respawning logic
- Player management (1-5 players)
- Movement system (speed ∝ 1/size)
- Collision detection (orbs and players)
- 60 FPS game loop
- Error handling throughout
- ~300 lines with comprehensive comments

### public/index.html (not shown in size)
**Frontend client implementation**
- PixiJS 6.5.2 rendering engine
- 800x600 canvas
- Socket.IO client
- Player rendering (green/blue circles)
- Orb rendering (red circles)
- Camera system (follows player)
- Mouse input handling
- Real-time state updates
- UI display (size, position, players)
- ~250 lines with extensive documentation
