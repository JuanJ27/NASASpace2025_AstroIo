PLAY NOW

3. Enter your name (alphanumeric, max 20 characters)

================================================================================
SERVER MANAGEMENT
================================================================================

CHECK IF SERVER IS RUNNING:
  lsof -i:3000 | grep LISTEN

START SERVER (if not running):
  node server.js

STOP SERVER:
  - Find PID: lsof -ti:3000
  - Kill: kill [PID]
  - Or: pkill -f "node server.js"

RESTART SERVER:
  pkill -f "node server.js" && node server.js

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "Cannot connect to server"
FIX: Check if server is running with: lsof -i:3000
     If not running: node server.js

ISSUE: "Port 3000 already in use"
FIX: Kill existing process: pkill -f "node server.js"
     Then restart: node server.js

ISSUE: "Game is full"
FIX: Server allows max 5 players. Wait for a slot to open.

ISSUE: Blank screen in browser
FIX: 1. Check browser console for errors (F12)
     2. Ensure server is running
     3. Try clearing cache (Ctrl+Shift+R)

ISSUE: Name won't submit
FIX: Name must be alphanumeric and max 20 characters.
     No special characters or emojis.
